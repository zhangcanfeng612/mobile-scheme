# mobile-scheme

> 移动端自适应布局解决方案

## Build Setup

``` bash
# install dependencies
$ yarn install

# serve with hot reload at localhost:3000
$ yarn run dev

# build for production and launch server
$ yarn run build
$ yarn start

# generate static project
$ yarn run generate
```

该项目使用NUXT构建 [Nuxt.js docs](https://nuxtjs.org).
